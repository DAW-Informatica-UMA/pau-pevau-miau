1. DESCRIPCIÓN DEL PROYECTO
--------------------------------------------------------------------------------
Este repositorio contiene un microservicio REST desarrollado para la asignatura 
Desarrollo de Aplicaciones Web (DAW) de la Universidad de Málaga (UMA). 

El sistema está diseñado para la gestión de la información del estudiantado 
en el marco de las pruebas PEvAU. Su funcionalidad principal incluye la 
importación de alumnos mediante ficheros estructurados (CSV/Excel), la 
generación de listados filtrables y ordenables, y la contextualización de 
datos por año académico.
